Magic help

Get awesome keyboard shortcut help 100% for free. Just press ? on any page to show help customised to the exact keyboard actions that are availale on this page.

Disabled shortcuts

Sometimes you want to show that an action is generally available, but not on this particular page.

<div data-mk-disabled="q" title="Logout of this application"></div>

Page shortcuts, eg search

Add a data-mk-* attribute to any element, and it will wire up a * event when that key is pressed. The magic help system will look at the text, value, or title of that element to generate the help text.

Press the ubiquitous / key to focus on the search

It would look for a label, or a title attribute on the element to auto generate the help text

<input type="search" data-mk-focus="/" placeholder="Look for stuff">

Another common example would be navigating to a next or previous page. The a 

<a data-mk-click="<" href="?prev">Previous page</a>

Lists actions

Often you have a main list of 'things', email in gmail, issues in github issues or jira, etc. First you want to be able to navigate them with increasingly ubiqutious j and k keys, and then when on is selected do stuff to it, like select it with space,

Nav to issues, toggle one as 'read' with r, or select

Note that magic keyboard is only forwarding key events into other events like click, and then handled by whatever logic you already have in place, so there is usually no code.

Page actions

This was inspired by Jira, just press . or , key to bring up a search box of actions that can be applies to this page.

You can declare an link, button, or form to be a page actions by adding the data-mk-action attribute to it, or to it's ancestor. So you could add it to the root element of a drop down menu to attach all the items in it.

<div data-mk-action>
<a class="btn" href="#">Send to cleaners</a>
<a class="btn" href="#">Send to dog minder</a>
</div>
Send to cleaners Send to dog minder

Custom shortcuts

You may already have some keyboard support, or the magic stuff just isn't magic enough.

But you still want these to appear in the magic help screen, so you can declare actionless things like this:

<div data-mk-noop="*" title="This does something real complex"></div>
